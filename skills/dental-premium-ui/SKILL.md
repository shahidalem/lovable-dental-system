# Dental Premium UI Skill

## Objective
Design premium, modern, non-generic dental clinic websites.

The output should feel like a custom $10k–$50k agency website.

Never generate template-looking websites.

## Design Inspiration
- ChatGPT
- Perplexity
- Apple
- Stripe
- Linear
- Framer premium templates

## Design Principles
- Premium aesthetic
- Strong whitespace
- Clean grid system
- Modern luxury feel
- Trust-building visuals
- Mobile-first

## Layout Rules
Always maintain:
- Clear visual hierarchy
- Balanced spacing
- Consistent alignment
- Section breathing room

Spacing scale:
- 8
- 12
- 16
- 24
- 32
- 48
- 72
- 96

Border radius:
16px to 28px

## Typography
Headings:
- Inter
- Manrope
- Satoshi
- Plus Jakarta Sans

Body:
- Inter
- DM Sans

Rules:
- Large hero headings
- Strong hierarchy
- High readability
- Avoid generic fonts

## Color Psychology
Preferred:
- White
- Soft Blue
- Navy
- Teal
- Cyan accents

Avoid:
- Neon colors
- Cheap gradients
- Harsh red
- Oversaturated palettes

## Animations
Allowed:
- Fade in
- Scroll reveal
- Hover transitions
- Soft parallax in hero

Duration:
300ms–700ms

Never:
- Over animate
- Use flashy effects
- Use bouncing animations

## CRO Rules
Every homepage should include:
1. Hero
2. Trust badges
3. Services
4. Doctor section
5. Testimonials
6. Gallery
7. FAQ
8. Strong CTA

Primary CTAs:
- Book Appointment
- Call Now
- WhatsApp

## Output Standard
Every website must feel:
- Premium
- Trustworthy
- Modern
- Custom-made
- Not AI-generated

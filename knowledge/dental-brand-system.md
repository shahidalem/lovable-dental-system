# Dental Brand System

## Core Philosophy
Every dental website must feel premium, trustworthy, modern and conversion-focused.

## Design Inspiration
- ChatGPT
- Perplexity
- Apple
- Stripe
- Linear

## Visual Principles
- Strong whitespace
- Clean layouts
- Premium typography
- Minimal but luxurious
- Non-generic UI

## Color Psychology
Primary:
- White
- Soft Blue
- Navy
- Teal

Avoid:
- Neon colors
- Cheap gradients
- Over-saturated red

## Typography
Headings:
- Inter
- Manrope
- Satoshi

Body:
- Inter
- DM Sans

## UX Principles
- Fast loading
- Mobile first
- Clear CTA
- Easy appointment booking

## Conversion Goals
Every page should optimize for:
- Appointment booking
- Call
- WhatsApp lead
- Trust building
